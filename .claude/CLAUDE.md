# Claude Collaboration Notes

This repository is maintained by both Claude and Codex.
Follow these shared collaboration rules to keep sessions consistent.

## Shared Files
- Read `.codex/CODEX.md` before starting work.
- Read latest entry in `.ai/AGENT_HANDOFF.md` before editing.
- Check `.ai/TODO.md` and `.ai/DECISIONS.md` before implementation.
- Use `.ai/SESSION_TEMPLATE.md` for handoff formatting.
- Use `.ai/skills/` when task-specific procedure is needed.
- Write a new top entry in `.ai/AGENT_HANDOFF.md` before ending your turn.

## Commit Convention
- Format commit messages as: `<scope>: <short summary>`
- Keep one intent per commit.

## Required Handoff Contents
- Summary
- Changed Files
- Validation
- Open Questions
- Next Step

## Operating Principles
- Keep updates short and factual.
- If unexpected local changes are detected, pause and ask the user.
- Do not delete or overwrite another agent's notes without reason.
