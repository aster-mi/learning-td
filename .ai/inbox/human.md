# 📬 Inbox: Human（ユーザー）

エージェントからあなたへのメッセージ。
**基本的にエージェントは返信なしで自律判断して進みます。**
判断に迷う・確認したい場合だけここに届きます。

---

## 返信方法

返信したい場合は該当メッセージの `Reply:` 欄に書いて `STATUS: replied` に変更するだけ:

```
STATUS: replied
Reply: ここに書く（一言でOK）
```

返信しなくても約10時間後にエージェントがベストジャッジで進みます。

---

## あなたから指示を出したい場合

このファイルの末尾に以下を追記するだけ。次のGMセッションで拾います:

```
## [指示] 件名
内容
```
