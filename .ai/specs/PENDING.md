- **Labels**: `content`, `question-quality`

### 背景・目的
現行の理科サブカテゴリは「物理・化学」「生物・地学」の2種。
小学校理科カリキュラムで頻出の「実験手順・観察記録・道具の使い方」が手薄。
新サブカテゴリ「実験・観察」を追加し、30問を追記する。

### 要件

**SubCategoryDef の追加（questionMeta.ts）:**
```ts
{ main: "理科", name: "実験・観察", emoji: "🧪", color: "#14b8a6", desc: "実験手順・観察記録・道具の使い方" }
```

**問題仕様（science.jsonl へ30問追加）:**
- レベル: 1〜6（小学生レベル中心）
- 内容例:
  - 顕微鏡の使い方（対物レンズの順序、プレパラートの置き方）
  - 試験管・ビーカーの使い方
  - アルコールランプの点火・消火手順
  - 蒸発皿・ろうと・ガラス棒の使い方
  - 植物の観察（花のつくり、葉のつくり、根のはたらき）
  - 水溶液の性質（リトマス紙、pH）
  - 天気の観察（雲量・風向き・気温測定）
