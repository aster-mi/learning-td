## SPEC-20260328-06: 算数「比・割合・速さ」サブカテゴリ新設 +40問

**Priority**: P2
**作成**: 企画＋調査 2026-03-28

### 背景・目的
算数カテゴリは425問あるが、Level7が28問・Level8が8問・Level9-10が0問と高難度が極端に薄い。
小学5-6年〜中学で頻出の「比・割合・速さ」テーマが現行問題にほぼ存在しない。

### 実装内容

**1. questionMeta.ts に新サブカテゴリ追加**
```ts
{ main: "算数", name: "比・割合・速さ", emoji: "⚖️", color: "#6366f1", desc: "比・百分率・速さ・単位換算" },
```

**2. math.jsonl に +40問追記（m482〜m521）**
- ID: m482〜m521
- sub: "比・割合・速さ"
- level分布: Level5×8問, Level6×8問, Level7×10問, Level8×8問, Level9×6問
- テーマ:
  - 比と割合（3:5の比の計算、比の値など）
  - 百分率（x%はいくつ？ 全体量からの割合算出）
  - 速さ・時間・距離（速さ＝距離÷時間 の応用）
  - 単位換算（km/h ↔ m/s、面積・容積換算）
  - 濃度・食塩水（食塩x%の溶液問題）

### 受け入れ条件
- `npm run quiz:validate` が通ること
- CategorySelect で「比・割合・速さ」サブが選択可能になること

## SPEC-20260328-07: 国語「文法・語彙」サブカテゴリ新設 +35問

**Priority**: P2
**作成**: 企画＋調査 2026-03-28

### 背景・目的
国語カテゴリは漢字・読み（185問）と熟語・慣用句（63問）の2サブのみ。
品詞・敬語・語彙（類義語・対義語）などの **文法・語彙ジャンルが完全欠如**。
Level9-10も0問のため高難度ユーザーのニーズに応えられていない。

### 実装内容

**1. questionMeta.ts に新サブカテゴリ追加**
```ts
{ main: "国語", name: "文法・語彙", emoji: "📚", color: "#8b5cf6", desc: "品詞・敬語・類義語・対義語・語彙" },
```

**2. language.jsonl に +35問追記（k299〜k333）**
- ID: k299〜k333
- sub: "文法・語彙"
- level分布: Level4×6問, Level5×6問, Level6×7問, Level7×7問, Level8×5問, Level9×4問
- テーマ:
  - 品詞分類（この文の〜は何詞？: 名詞・動詞・形容詞・副詞・接続詞など）
  - 敬語の種類と使い方（丁寧語・尊敬語・謙譲語の区別）
  - 類義語・対義語（「積極的」の対義語は？）
  - 語彙・送り仮名（「表す」「現す」の使い分けなど）
  - 文の組み立て（主語・述語・修飾語の関係）

### 受け入れ条件
- `npm run quiz:validate` が通ること
- CategorySelect で「文法・語彙」サブが選択可能になること
