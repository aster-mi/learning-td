# SKILL: 仲間ユニットバランス調整

## 目的
- 強すぎる/弱すぎるユニットを是正し、編成の多様性を維持する。

## 前提
- このSkillは「数値調整」が中心。  
  ただし、見た目が名前と乖離している場合は `SKILL_UNIT_ADD.md` に従ってデザインも修正する。

## 手順
1. 課題を分類する。
- 火力過多
- 耐久過多
- 射程過多
- 役割不明（採用理由がない）
2. 1回の調整対象は少数に絞る（3〜8体推奨）。
3. `src/data/unitCatalog.ts` の値を調整する。
4. `npm.cmd run build` を実行する。
5. 実戦で確認する。
- コストに見合う強さか
- 既存の役割を潰していないか

## ルール
- 高レアでも欠点は残す。
- 長射程ユニットは耐久か速度を抑える。
- 高速ユニットは耐久か単発火力を抑える。
- 完全上位互換を作らない。
