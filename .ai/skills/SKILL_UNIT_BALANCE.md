# SKILL: 仲間ユニットバランス調整

このSkillは、ユニット性能の再調整を行うときに使う。

## 目的
- メタ固定化を防ぐ
- 使われないユニットを減らす
- ステージ難易度との整合を維持する

## 実施手順
1. 対象ユニットと問題を明確化する（強すぎる/弱すぎる/役割不明）。
2. 調整方針を決める。
- 火力過多: `atk` を下げる、または `atkInterval` を伸ばす
- 耐久過多: `hp` を下げる、または `cost` を上げる
- 器用貧乏: 長所を1つ伸ばし、短所を1つ明確化する
3. `src/data/unitCatalog.ts` のみで完結できる調整を優先する。
4. 影響が大きい場合のみ、段階的に2回に分けて調整する。
5. `npm.cmd run build` で検証する。

## 調整ガイド
- 単発火力を上げる場合は攻撃間隔を長めにする
- 射程を伸ばす場合は耐久か速度のどちらかを下げる
- コスト10差で体感差が出るようにする
