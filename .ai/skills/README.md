# Skills カタログ

このディレクトリは、プロジェクト運用の標準手順（Skills）をまとめたものです。  
基本的に日本語で更新します。

## 既存Skills
- `SKILL_QUESTION_ADD.md`: 問題追加の手順
- `SKILL_QUESTION_QC.md`: 問題品質チェック
- `SKILL_GIT_WORKFLOW.md`: Git運用ルール
- `SKILL_DEPLOY_PAGES.md`: GitHub Pagesデプロイ手順
- `SKILL_RELEASE_CHECK.md`: リリース前チェック
- `SKILL_AGENT_HANDOFF.md`: Codex/Claude 引き継ぎ

## Unit Skills
- `SKILL_UNIT_ADD.md`: 仲間ユニット追加の標準手順
- `SKILL_UNIT_BALANCE.md`: 仲間ユニット調整の標準手順
- `../UNIT_POLICY.md`: ユニット設計ポリシー

## 使い方
1. 対象作業に対応するSkillを開く
2. 手順どおりに実行する
3. 実施結果を確認する
4. 必要なら `.ai/AGENT_HANDOFF.md` に共有する
