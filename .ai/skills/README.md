# Skills カタログ

このディレクトリは、このプロジェクト専用の作業スキル集です。  
基本言語は日本語で運用します。

## 一覧
- `SKILL_QUESTION_ADD.md`: 問題追加・編集手順（JSONL運用）
- `SKILL_QUESTION_QC.md`: 問題品質レビュー手順（文面/選択肢/整合性）
- `SKILL_GIT_WORKFLOW.md`: Git運用ルール（コミット/Push/競合回避）
- `SKILL_DEPLOY_PAGES.md`: GitHub Pagesデプロイ確認手順
- `SKILL_RELEASE_CHECK.md`: リリース前チェックリスト
- `SKILL_AGENT_HANDOFF.md`: Codex/Claude 引き継ぎ運用手順

## 使い方
1. 実施したい作業に対応する Skill を開く
2. 手順どおりに実行する
3. 完了条件を満たしているか確認する
4. 必要なら `.ai/AGENT_HANDOFF.md` に記録する
