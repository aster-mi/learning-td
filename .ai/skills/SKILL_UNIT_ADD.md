# SKILL: 仲間ユニット追加

このSkillは、仲間ユニットを安全に追加するときに使う。

## 入力
- 追加したいユニット数
- 役割（前衛/高速/射程/支援など）
- 想定レアリティ

## 実施手順
1. `src/data/unitCatalog.ts` にユニットを追加する。
- `id` は英小文字で一意
- `series` は既存シリーズを優先（必要なら新規）
- `desc` はゲーム内で意味が通る短文
2. 描画方針を決める。
- 既存レンダラーで十分なら `emoji` fallback を使う
- 専用見た目が必要なら `src/components/renderers/*` に追加し `index.ts` へ登録
3. バランスを仮調整する。
- `cost` と `hp/atk/atkInterval/speed/range` のトレードオフを作る
- 役割が既存と被る場合は長所短所を明示する
4. 検証する。
- `npm.cmd run build`
- パーティ選択画面で表示確認
- 実戦で召喚と攻撃挙動を確認
5. 変更説明を残す。
- 追加理由
- 役割
- 弱点

## チェックリスト
- `GACHA_POOL_UNITS` に自然に含まれる（`INITIAL_UNITS` で除外していない）
- 1体でも `id` 重複がない
- 高レアが低レアの完全上位互換になっていない
